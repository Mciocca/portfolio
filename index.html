Embr

×

Embr was created as a ticketing system for our marketing department and to automate the creation of marketing materials. I was the lead developer on the project and was responsible for making our technology choices and building out key features.

Embr is a single page application built with React.js and Flux on the frontend and a Laravel based restful api on the backend. It also features real time chat and notifications by using an Express.js webserver listening to Redis events fired by Laravel and Socket.io.

Embr is heavily used by our sales team and we are planning to add features to meet the needs of other parts of our company.

Created with:

  • PHP 7 with Laravel
  • React, React Router, and Flux
  • ES6 (webpack and babel)
  • Mocha, Enzyme, and Expect.js for testing
  • Redis
  • Node.js (Express web server)
  • Socket.io
  • Mandrill
  • Hosted on an Ubuntu Digital Ocean VPS

Clutch Fantasy Baseball

×

This is the mobile web app that will be released as a companion to the upcoming Clutch fantasy baseball card game. It's used by players to keep track of many different in game stats.

The app is built with Vue.js for the frontend. The app allows users to save games using Express.js and MongoDB. Authentication is done using Passport.js and it's facebook authentication strategy.
